## Installation

Packages required prior to installing DreamAI
```
require("pamr")
require("cluster")
require("survival")
require("randomForest")
require("missForest")
require("glmnet")
require("foreach")
require("itertools")
require("iterators")
require("Matrix")
require("devtools")
```

Install DreamAI using the R package "devtools"
```
install_github("WangLab-MSSM/DreamAI/Code")
```
